Vielen Dank, dass Sie sich für SchuleIntern entschieden haben.

 Sie können hier alle nötigen Verträge online zeichnen.

Wie funktioniert die elektronische Zeichnung?

  • Wählen Sie den Vetrag aus, den Sie zeichnen möchten.
  • Lesen Sie sich den Vetragstext sorgfältig durch.
  • Geben Sie am Ende den Namen des Zeichnungsberechtigten und die dienstliche E-Mailadresse des Zeichnungsberechtigten ein.
  • Klicken Sie auf "Zeichnungscode anfordern".
  • Sie erhalten einen Zeichnungscode per E-Mail zugesandt.
  • Geben Sie den Code ein und klicken Sie auf "Zeichnen".
  • Sie erhalten eine Kopie des Vertrages hier zum Download und per E-Mail als PDF zugesendet.
  • Die Veträge sind von Seiten Spitschka IT Solutions bereits gezeichnet. Sie können sofort loslegen.
