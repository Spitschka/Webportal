Ich möchte die SMS Funktion aktivieren. Mir ist bewusst, dass dadurch weitere Kosten entstehen.
Hinweise:
  • Wird die SMS Funktion aktiviert, können Benutzer ihre SMS im Benutzerprofil hinterlegen.
  • Welche Benutzergruppen ihre Handynummer hinterlegen können, können Sie in den Einstellungen festlegen. (Schüler, Eltern oder Lehrer)
  • Die Nummer kann z.B. für schnelle Nachrichten (z.B. Unterrichtsausfall wegen Wasserschaden) verwendet werden.
  • Die Handynummern sind für andere Personen nicht sichtbar.
  • Weiterer Nutzen der SMS sind in Zukunft: Zwei-Faktor-Authentifizierung
  • Benachrichtigungen über neue Vertretungen etc. können nicht per SMS erfolgen. (zu teuer)
Kosten:
  • Der SMS Versand ist nicht im Portalpreis enthalten. Die Abrechnung erfolgt nutzungsabhängig.
  • Eine einzelne SMS kostet € 0,085 pro Stück (zzgl. Mehrwertsteuer)
  • Das hinterlegen einer SMS benötigt eine SMS Nachricht zur Verifizierung pro Benutzer.
  • Es muss ein Guthaben im Voraus gebucht werden.
  • Sie können hier einfach Guthaben aufladen, wie Sie möchten. Sie erhalten dann in den nächsten 1-2 Wochen eine Rechnung über den Betrag.
