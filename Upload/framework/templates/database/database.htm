Datenbanken für den IT Unterricht

Hier können MySQL - Datenbanken für den Unterricht oder für Projekte verwaltet werden.

Der Server ist nicht für persönliche Daten gedacht, da er nicht speziell abgesichert ist. Es handelt sich um einen "Spielserver" zum Erlernen von SQL.

Jeder Datenbank können dazu beliebig viele Benutzer zugeordnet werden. Bei der Zuordnung wird angegeben, welche Rechte der Benutzer haben soll.

Es können beliebig viele Benutzer und Datenbanken erstellt werden.

Auf die Datenbanken kann hier zugegriffen werden: http://databaseserver.schule-intern.de/ (PHPMyAdmin)
