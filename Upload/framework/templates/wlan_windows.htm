{$this->header} Konfigurationsanleitung für Windows

Auf dieser Seite wird erklärt, wie man am privaten Laptop mit Windows 7 das Wlan des Gymnasium Pullachs nutzen kann.
Bei andere Versionen von MS Windows funktioniert die Installation ähnlich.

Bitte alle Schritte genau lesen, verstehen und befolgen!

  1. Zuerst klickt man auf das kleine Netzwerk-Symbol im Systemtray der Taskleiste (unten rechts am Bildschirm)



  2. Es öffnet sich ein kleines Fenster. Dort auf Netzwerk- und Freigabecenter öffnen klicken



  3. Im nun folgenden Fenster klickt man auf Drahtlosnetzwerke verwalten…



  4. …dann auf Hinzufügen



  5. Im folgenden Fenster wählt man Ein Netzwerkprofil manuell erstellen



  6. Als Netzwerkname trägt man gympu ein.
    Der Sicherheitstyp ist WPA2-Enterprise.
    Die Verschlüsselung bleibt auf AES



  7. Nach dem Klicken auf Weiter gelangt man zu folgendem Fenster, in dem man auf Verbindungseinstellungen ändern klickt.



  8. Hier wählt man zuerst den Reiter Sicherheit aus.



  9. Dort wählt man Einstellungen



  10. Das Häkchen bei Serverzertifikate überprüfen darf nicht gesetzt sein!
    Anschließend klick auf Konfigurieren...



  11. Das Häkchen in dem aufpoppenden Fenster muss entfernt werden, danach klick auf Ok.
    In dem verbleibenden Fenster auch auf Ok klicken.



  12. Jetzt Erweiterte Einstellungen öffnen...



  13. …dort den Haken bei Authentifizierungsmodus angeben setzen und den Modus auf Benutzerauthentifizierung umstellen.



  14. Jetzt können alle Fenster mit Klick auf Ok geschlossen werden.

    Rechts unten im Systemtray erscheint folgendes Popup, auf das man kickt:



  15. In das nun erscheinende Fenster gibt den Benutzernamen und das Passwort ein, das man auch an den Schulrechnern verwendet.



  16. Jetzt sollte sich Windows mit dem Wlan verbinden. Ob es geklappt hat, sieht man daran, dass das Netzwerksymbol aus mehreren weißen Balken in unterschiedlicher Größe besteht.
    Möglicherweise ist anfangs ein kleines gelbes Dreieck mit Ausrufezeichen zu sehen, das sollte aber bald verschwinden.
  17. Wenn es Probleme geben sollte, bzw. wenn die Anleitung nicht zum Erfolg führt, wendet Euch natürlich an die Systemadministratoren (Leo Fellermayer Fl, Christian Spitschka, Tobias Heimbeck Hk)
  18. Viel Spaß beim Surfen!!
