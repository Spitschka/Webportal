Klassenleitung {$this->klasse} - Drucken der Bögen

An dieser Stelle werden die Beobachtungsbögen gedruckt.
Bitte achten Sie auf folgendes:

  • Überprüfen Sie alle Namen! Die Namen, die im Feld angegeben sind, werden so auf den Bogen gedruckt.
  • Für jede Frage ist die fertige Beurteilung angezeigt.
  • Geben Sie bitte Ihre Signatur an (wie es auf dem Bogen ausgedruckt wird.)
  • Geben Sie bitte die Klassenbezeichnung an (hier können Sie die Zweigergänzung (z.B. _f) entfernen.)
  • Nach dem Absenden erhalten Sie eine PDF zum Download, die alle fertigen Bögen enthält.
  • Drucken Sie bitte die fertigen Bögen auf dem ausgegebenen Papier mit dem Farblaserdrucker aus.
  • Unterschreiben Sie die Würdigungen und fertigen Sie eine Kopie mit dem Wasserzeichen "KOPIE" am Kopierer an.
