Über das Home-Use-Program haben Lehrer die Möglichkeit eine Lizenz Microsoft Office Professional Plus 2016 für Windows oder für Mac zu bestellen.

Es handelt sich dabei nicht um Abo-Lizenzen, wie bei Office 365, sondern um klassische Lizenzen wie im Einzelhandel. Es ist nur zur Produktaktivierung eine Internetverbindung erforderlich.

Die Lizenz ist unbegrenzt gültig. Beachten Sie aber: Die müssen die Software deinstallieren, wenn Sie die Schule verlassen! Dies ist von Ihnen selbst sicherzustellen, da es von uns nicht überprüft werden kann.

Eine Weitergabe oder ein Verkauf der Software ist nicht gestattet.

Die Kosten betragen 13,95 pro Lizenz, die von Ihnen direkt an Microsoft gezahlt werden. (Paypal, Kreditkarte, etc...)

Pro Person kann nur eine Lizenz (Windows oder Mac) gekauft werden.

Die Vorgehensweise ist:

  1. Gehen Sie auf die HomeUseProgram Seite von Microsoft: http://www.microsofthup.com/
  2. Dort werden Sie nach Ihrer geschäftlichen E-Mailadresse ($username@$maildomain) und einem Programmcode gefragt.
    Geben Sie als Porgrammcode folgenden Code an: $hupCode
  3. Folgen Sie den Anweisungen auf der Seite.
