Zweifaktorauthentifizierung mittels Mobiltelefon

Die Zweifaktorauthentifizierung ist aktiviert.
Wenn Sie die Zweifaktorauthentifizierung deaktivieren wollen, dann geben Sie bitte den aktuellen Code hier ein:

Sollten Sie keinen Zugriff mehr auf ihr Gerät haben, dann kontaktieren Sie bitte den Administrator der Schule.

Die Deaktivierung kann bis zu 10 Sekunden benötigen! Wenn Sie die Zweifaktorauthentifizierung aktivieren, können Sie auf bestimmte Seiten nur noch durch die Eingabe eines Codes aus der TOTP App eintreten. Auch ist eine Kennwortänderung nur noch durch Eingabe des TOTP Codes möglich.

Zur Aktivierung der Zweifaktorauthentifizierung laden Sie bitte eine der folgenden Apps herunter:

Android
  • Google Authenticator
  • andOTP
iOS
  • Google Authenticator
  • FreeOTP Authenticator
Scannen Sie in einer dieser Apps bitte folgenden QR Code richten Sie damit die APP korrekt ein.



ACHTUNG:
Sie können diesen QR Code nach der Aktivierung NICHT mehr abrufen. Sichern Sie sich bitte daher diesen Code, indem Sie ihn jetzt ausdrucken und an einer sicheren Stelle verwahren. QR Code ausdrucken / Als PDF speichern

Die Aktivierung kann bis zu 10 Sekunden benötigen!

Achtung - Uhrzeit muss korrekt eingestellt sein

Die Uhrzeit auf Ihrem Endgerät muss korrekt eingestellt sein, damit sie die gleiche Uhrzeit, wie auf dem Server eingestellt ist.

Die Aktuelle Uhrzeit auf dem Server (Zum Zeitpunkt des Ladens der Seite - Neu laden)
